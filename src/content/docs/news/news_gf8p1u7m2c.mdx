We've just released **ClassFlow v1.0.0** — a milestone update packed with performance, polish, and powerful new capabilities.

## What's New

This release brings search across lists, bulk actions, a guided tutorial, and significant accessibility improvements for a smoother experience on every device.

## Highlights

- Search bars for subjects, books, and timetable lists
- Multi-select for bulk delete/edit of subjects and books
- Interactive tutorial for first-time users
- Improved screen reader support, semantic labels, and contrast ratios
- Polished page transitions with subtle micro-interactions
- iCloud/Google Drive integration for automatic backups
- Stricter input validation and clearer error messages
- Action history to help recover accidental deletions
- Archive old subjects/books instead of permanently deleting
- Home screen highlights today's schedule
- Optional notes on lessons for homework and reminders
- Summary cards: lesson count, next lesson, and free time
- Mark lessons as cancelled, modified, or rescheduled
- User accounts with cloud storage for cross-device sync
- Custom server configuration for self-hosted or official cloud servers
- Offline-first: queue local changes and sync when online
- Smart merge when data changes on multiple devices
- Export timetable to PDF, iCal, or CSV formats
- New color themes beyond light/dark mode
- Drag & drop to reorder lessons and subjects
- Save recurring lesson patterns for quick scheduling
- Explicit break/lunch periods
- Better timezone handling for traveling users

## Update Now

Open AltStore and update ClassFlow to **v1.0.0** to get all these improvements. As always, your feedback keeps ClassFlow getting better — thank you for being part of the journey!
